Repeat

Syntax

rep [n]
(tab)statements
(tab)...

 

Can be single line:

rep([n]) statements

 

Parameters

n - number of times to repeat. If omitted, repeats forever.

 

Remarks

Repeatedly executes statements.

 

Use break to exit loop. Use continue to skip following statements.

 

See also : for, foreach.

 

Examples

rep 10
	bee
	1

rep(10) bee; 1

rep
	i+1
	if(i>10) break
